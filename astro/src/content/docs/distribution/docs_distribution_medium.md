Medium distribution uses Medium's import flow, not direct API posting.

## Prerequisites

- Medium account
- Publicly accessible Whitepapper paper URL

## Import flow

1. Copy paper URL from Whitepapper
2. Open Medium stories page
3. Click **Import a story**
4. Paste Whitepapper URL
5. Review imported draft
6. Publish from Medium

## Canonical behavior

Medium import flow keeps the source URL relationship, so canonical attribution can remain aligned with your Whitepapper source URL.

## Limitations

- Formatting cleanup can be needed after import
- Import requires public URL accessibility
- No automatic sync for future edits
