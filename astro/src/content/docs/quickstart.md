# Quickstart

Create your first project and publish quickly.

## 1. Create a project

- Go to Dashboard.
- Click New Project.
- Enter a project name and save.

## 2. Open API creation

- After the project is created, open that project workspace.
- Click API in the project sidebar.
- Choose Create API and pick the endpoint type you need.

## 3. Add your first endpoint

- Add route path and method.
- Set authentication and request schema.
- Save, then run a test request.

## 4. Publish and verify

- Publish from the project toolbar.
- Copy the generated endpoint URL.
- Verify responses from your app or API client.
